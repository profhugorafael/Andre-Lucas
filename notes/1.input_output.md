# Input e Output

## Output

```py
nome = 'Andre'
print('Meu nome eh' + nome)

# interpolar
idade = 23
print(f'minha idade eh {idade}')

calculo = 12.125 * 100.45
resultado = f'R$ {calculo:.2f}'

print(resultado)
```

## Variaveis

- `int`: inteiro
- `float`: real / com ponto
- `str`: texto / string
- `bool`: verdadeiro / falso
